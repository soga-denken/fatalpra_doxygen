What is FATALPRA?

FATALPRA stands for "Fatalpra is A TrustAble program for a Level 1 Probabilistic Risk Assessment", which provides basic functionalities to perform probabilistic risk assessment for Nuclear Power Plants and any other industrial applications that require PRA.
FATALPRA includes fundamental fault trees and event trees manipulation and algorithms to perform risk assessment. FATALPRA itself does not provide GUI, but one can build such a software upon FATALPRA frameworks.
