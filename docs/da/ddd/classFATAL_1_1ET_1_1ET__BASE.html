void FATAL::ET::ET_BASE::auto_replace ( std::vector< boost::shared_ptr< ET_BASE >> &  sequences)
virtual

this function automatically replace if a null branch with sequence auto replacement rules are below:

  1. if both success and failure branches are null, then this branch is replaced with sequence
  2. either one of branch is null, only the null branch is replaced.
    Warning
    Do not call this function on IE.
